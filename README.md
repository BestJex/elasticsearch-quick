# springboot-jest 

SpringBoot使用jest替代官方sdk的使用方法

简单易用，无需担心版本不兼容的问题

配置好es的链接参数，直接运行项目，程序会自动的创建索引、mapping并且会插入几个文档供接下里的CURD使用
